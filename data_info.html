Data info

You can import aggregated or individual data. In both cases you have to satisfy the predefined format. If dates are not "date" in Excel but "string", you have to specifiy the format (default: YYYY/MM/DD).

An aggregated data file contains three columns

date You can specify the format in the field "Date format"
nhos Cumulative counts of hospitalized patients current
nicu Current number of occupied beds in ICUs

An individual data file contains at least one sheet with the first column named "No interne*". The other mandatory columns are

arrivee_hopital Date of hospitalization (date format or empty)
debut_soins_intensifs Date of entry into intensive care (date format or empty)
fin_soins_intensifs Intensive care discharge date (date format or empty)
