Data info

You can import aggregated or individual data. In both cases you have to satisfy the predefined format. If dates are not "date" in Excel but "string", you have to specifiy the format (default: %d.%m.%Y).

An aggregated data file contains three columns

date You can specify the format in the field "Date format". Default: %d.%m%Y
nhos Cumulative counts of hospitalized patients current
nicu Current number of occupied beds in ICUs
